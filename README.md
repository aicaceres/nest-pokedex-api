## Description

Pokemon API clone bootstrapped with [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Getting Started

1. Clone the repository
2. Run
```
npm install
```
3. Install Nest CLI
```
npm i -g @nestjs/cli
```
4. Database set-up
```
docker-compose up -d
```
5. Populate the database
```
http://localhost:3000/api/seed
```
## Stack
* Nest
* MongoDB
* Docker
